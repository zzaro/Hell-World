# PortE

<img src = "./ScreenShots/icon.png" width="20%">  
인천공항의 내부 안내, 항공권 검색, 나의 항공권 정보에 따른 탑승구 안내 및 대기자 현황을 제공하는 협업 프로젝트

## 활용기술
- Kotlin
- MVVM 패턴
- 공공 API활용
- Retrofit2
- Firebase
